## 🔥 ZORUNLU UNSURLAR MİMARİSİ (MASTER)

### Gate Sistemi

| Gate | Ad | Açıklama |
|------|-----|----------|
| GATE_1 | Takip Oluşturma | Dosya açılır, taraflar hazırlanır. Adres opsiyonel. |
| GATE_2 | Örnek 1 Üretimi | Takip talebi üretilecekse zorunlu alanlar kontrol edilir. |
| GATE_3 | Tebligat | Tebligat çıkarılacaksa adres zorunlu. |
| GATE_4 | UYAP Gönderimi | Örnek 1 + evrak + imzacı avukat zorunlu. |

### Takip Tipleri

- ILAMSIZ_GENEL (Fatura/Sözleşme/Cari)
- KIRA_ALACAGI (İlamsız kira)
- TAHLIYE (İlamsız tahliye)
- KAMBIYO_CEK (Çek)
- KAMBIYO_SENET (Senet/Bono)
- ILAMLI_GENEL (Genel ilamlı)
- ILAMLI_NAFAKA (Nafaka)
- ILAMLI_DOVIZ (Döviz alacağı)
- REHIN_IPOTEK (Rehin/İpotek)
- ISCILIK (İşçilik alacağı)

### Politikalar

```yaml
check_policy:
  bad_check_compensation_default_on: true  # Çek tazminatı varsayılan AÇIK
  bad_check_compensation_rate: 0.10        # %10

address_policy:
  allow_case_creation_without_address: true   # Gate1'de adres opsiyonel
  block_service_without_address: true         # Gate3'te adres zorunlu
  create_task_when_address_missing: true      # Adres eksikse task oluştur

interest_policy:
  require_post_filing_interest_by_default: true  # Takip sonrası faiz varsayılan
```

## 📋 API ENDPOINT'LERİ

### Validation Gate (/api/validation-gate)
- POST /:caseId/validate/:gateId - Belirli gate için validasyon
- POST /:caseId/validate-all - Tüm gate'leri validate et
- POST /:caseId/case-creation - Gate 1 validasyonu
- POST /:caseId/ornek1-generation - Gate 2 validasyonu
- POST /:caseId/service-of-process - Gate 3 validasyonu
- POST /:caseId/uyap-integration - Gate 4 validasyonu
- GET /rules - Validasyon kurallarını getir
- GET /policy?key=xxx - Politika değerini getir
- GET /check-compensation-info - Çek tazminatı bilgisi
- GET /address-suggestions - Adres önerileri

### Exchange Rate (/api/exchange-rate)
- GET /rates - Tüm kurlar
- GET /rate/:currency - Belirli kur
- GET /convert - Döviz çevirme
- GET /historical - Geçmiş kur
- POST /refresh - Kurları güncelle

### Tariff (/api/tariffs)
- GET / - Tüm tarifeler
- GET /active - Aktif tarife
- GET /:year - Yıla göre tarife
- POST /:year/create - Yeni tarife
- PUT /:year - Tarife güncelle
- DELETE /:year - Tarife sil
- POST /:year/import - JSON import
- GET /:year/export - JSON export

### Gazette Watcher (/api/tariffs/gazette)
- GET /notifications - Bildirimler
- GET /notifications/unread - Okunmamış bildirimler
- POST /notifications/:id/read - Okundu işaretle
- POST /notifications/read-all - Tümünü okundu işaretle
- POST /check - Manuel kontrol
- GET /status - Servis durumu

### Template Engine (/api/template-engine)
- POST /takip-talebi - Takip talebi oluştur
- POST /odeme-emri - Ödeme emri oluştur
- POST /icra-emri - İcra emri oluştur
- POST /haciz-tutanagi - Haciz tutanağı oluştur
- GET /templates - Şablon listesi

### PDF (/api/pdf)
- GET /takip-talebi/:caseId - Takip talebi PDF
- POST /generate - Genel PDF oluştur
- POST /generate-table - Tablo PDF oluştur
